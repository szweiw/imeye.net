寒门难出贵子

发表于 2017-07-06 | 分类于 他山之石

出自色影无忌网交流论坛，作者 钢种镬子（愤怒的秃子）

　　周围有几个工薪家庭的蠢货家长，纷纷给子女报一些小提琴钢琴班，凡是向我咨询意见的，我都建议学个口琴吉他算了，但没有一个听的，他们对于艺术的崇拜，有点类似坛里的佛学粉和国学粉，自己不懂的东西就觉得神秘，有无上的憧憬。他们这辈子不可能的事情，赌自己的小孩能帮他们yy。

　　为什么不建议蠢货家长给小孩报钢琴班？
　　首先钢琴是种技术，耗费大量的时间资源。工薪家庭不具备这个资源。为什么？先不说一星期雷打不动的上培训班，单单家里练习，99%的家庭不具备这条件。
　　想起以前看的一本书，说为什么划船比赛总是剑桥牛津之间？一方面是传统，一方面确实也只有剑桥牛津能组织起来高水平的划船比赛。这个关于选手的素质。也有很多人组织过划船训练，从比较差的学校录取选手，但教练发现这些选手连准时参加训练都做不到，不是请假就是迟到，要知道划船特别讲究选手之间的适应和配合，今天缺个你，明天缺个他，根本练不起来。和往常的寒门更能吃苦的概念相反，倒是伊顿公学这类纨绔子弟学校更能培养准时的学生，也更会遵循规则，更会配合同伴，当然，也更能吃苦。
　　而准时训练有两个前提，第一是确实能抽出这个时间，第二是自律性。寒门生计为重，就算不是为了生计，空余出来的时间也有更多的代办事项，比如爹妈为了户口要跑一趟老家，为了升迁要下班去领导家活动一下，随时耽误送子女去钢琴课。而这类家庭的父母，本身就缺乏自律性，不要奢望儿女耳闻目染，孩子同样难以管束自己。
　　隔壁有个帖子讲减肥的，减肥不是单独的一个动作，并非是你日常生活不变，额外添加一个动作。而是转变整个生活状态，包括合理饮食，抽出时间锻炼。甚至转变你这个人本身，你得自律起来，你出差半夜回家也最好别给自己借口，该举铁举铁，该跑步跑步。你应酬时也克制眼前的那罐黑啤，克制眼前的那口生蚝。绝大多数就败在自律二字上。
　　练钢琴同样如此，并不是你工薪家庭什么都照常，仅仅多一个练钢琴的动作，而是整个家庭环境得转变成适合练钢琴的家庭。但这个是天方夜谭。
　　更何况子女到了练钢琴的年纪，基本心性也定型了，爹妈的影响几乎牢不可破，搭个乐高都会搭的不耐烦，还练什么钢琴？最后的结果99%是爹妈训斥责打，小孩涕泪横流，最后家无宁日，爹妈看着不争气的孩子长吁短叹。怪谁？其实都怪你们自己。
　　自律的客观条件和主观能力，99%的工薪家庭都不具备。如果具备这个主客观素质，也绝对不会是这种现状。为什么减肥的大多数失败？缺乏自律的人，别说减肥钢琴，做什么不失败？

　　其次，钢琴毕竟是艺术，就爹妈听羊爱上狼的审美，不要指望孩子有什么艺术欣赏水准。
　　当然，不排除朗朗这种天纵英才，但这是万分之一的概率。如果有人寄希望于概率，我建议买彩票比送孩子练钢琴更靠谱。
　　有人会说，又不指望孩子获奖或者靠钢琴吃饭，只是要熏陶一下。最烦的就是熏陶两个字，就你这种家庭，能熏陶出什么？

　　最后，是钢琴耗费太多脑力，从小到大周围很多玩艺术的，各种艺术从业者，但我观察一下，各门艺术造诣高的，智力都不错，唯独学钢琴的，还没见过一个脑子好使的。记谱和学那些复杂的和声对位，耗费太多智力，能对和声对位有天赋的，数学水平肯定不会差，有这水平，还不如好好读书。

关于数字货币

发表于 2017-07-02 | 分类于 他山之石

关于数字货币

摘自水煮论坛，湖仙
就在看到论坛上湖仙这个帖子之前，我都还认为，没有国家信用的背书，没有暴力机关和经济组织的维持，数字货币会像君子兰、普洱茶、藏獒一样被恶炒之后抛弃……

从“游戏币”到“确实的货币”

中间是一个繁复的变化过程
并不是“产出的币”立刻就是货币
在现实中“得到承认”“得到使用”“得到交流”或者“资产置换的背书”
才算真实的货币
之前，怎么玩都是游戏币，不管多少价格买卖
之后，无论怎么糟蹋，都是货币，不管多少价格买卖
四年前，比特币无论怎么说，也是游戏币
现在，不管怎么看待，他都是货币
这是事实

只不过，从一个事实，到大众接受事实，需要一个过程
就像十五年前，没人看的清：房子，其实背后的土地资源是可以用货币计价的
当年说房子贵，是因为建筑成本1000元，你没有理由卖1500元
土地价值的货币倾向，并没有被大众理解和发现
直到现在，最后被接受，需要一个过程

所以绝大多数人，仍然把比特币和其他的虚拟货币，混为一谈
因为并没有发现：比特币已经是货币了
其他的绝大多数根本就没有想要成为过真正的货币，仅仅是沉醉于游戏币的圈子里
少数的，正在从游戏币圈子里争取成为货币，在路上
无论是已经成为货币的比特币，还是少数在路上有追求的游戏币
都还没有被大众完整的理解和认同

这是事实，因为这是事实，这才是价值。

世界在变革

而且变革的速度，越来越快

关于数字货币，你只要知道：
从你知道网络，到普及网购，需要多久？
那么，从你知道数字货币，到普及数字货币，需要时间一定比网购的时间短
你只要知道这个，就行了。

有几个要点 一定要讲清楚

能不能理解看个人了：

  1. 以比特币为代表的数字货币，死不了了
  2. 数字货币天生就是世界各国政府的敌人
  3. 世界各国政府都试图把数字货币扼杀过
  4. 但是有些国家政府慢慢发现：强大国家和强大国家的货币也是自己的敌人
  5. 有些逐渐可能边缘化的国家选择支持数字货币，只是因为数字货币是强大国家和强大国家货币的敌人
  6. 比特币死不了了，那么至少有一批数字货币会成为现实货币的备用品，目前看比特币和以太币是确认死不了的

所以，我同样建议大家去买币、甚至炒币，只不过我觉得最安全的，就是挖矿
愿意的一起来，当然条件成熟时候，我可不止是准备挖矿了。

我这六条，请仔细看，没有一条不是想很久才明白的
也许是因为我笨

另外说一些现象：

比特币，全球开采的平均电费是1.5万
现在价格大约1.8万
中国曾经占比特币产区的75%以上
年初是60%，现在据说是40%

也就是说，你看我们在疯狂上机器、增加算力
但是我们还是比国外上的少，少得多得多得多
也说明实际上，现在比特币的价格，只是基础成本+流通溢价
并没有明显附加“货币”的附加值
货币的附加值需要在一段时间以后慢慢体现

目前看，今年数字货币的炒作基本结束了
我个人认为，后面价格会沉寂一段时间，不排除按照月计算。

关于数字货币

我个人认为这个东西（包括比特币、以太币、莱特币等等等等）
最终会成为现实货币的备用品，相当于保险

保险将会成为总资产的大约5%～10%份额
时间大约是５年，大约可以达到全球总资产的3%以上
现在总资产大约是250万亿美元

这里面，一定有很多币死在路上，也有很多币根本就是骗局
但是也一定有一些，百倍千倍的上涨
甚至更高
我没有本事炒币
我只有本事看谁不死

最后，关于纸币

纸币的印刷、成本（产生+流通+回收成本），大约是面值的4%～20%
各国都不一样
再说一下以前的铜钱，铜钱的制造成本大约是币值的30%～55%
数字货币的产生成本，最后一定会低于目前纸币这个比例

货币在变革的过程中，自身的成本在逐渐减小。

我想说的，都说完了

关于共享单车的想法

发表于 2017-06-04 | 分类于 胡思乱想

关于“共享单车”的想法

1. 是共享单车还是公租单车？

深圳街头，大约半年前，突然出现了很多“共享单车”，橘色的、蓝色的、黄色的、绿色的，仿佛一夜之间就充斥着各条大街小巷，无论走到哪里，都避不开它们。在我原本的意识里，共享的涵义，是每个人将自己的物品拿出来，大家免费交换，这才叫共享，而现在市面上出现的这些小黄小蓝们，产权属于运营企业、需要收取押金和租金，跟共享精神相差甚远，似乎不能称为共享单车，应该叫做公租单车才对啊。
不过，约定俗成的力量是强大的，既然大家都叫他们为共享单车，我也只有跟着大家这么叫了。

2. 共享单车出现后，政府、企业、个人三者的责任；

共享单车出现后，最傻眼的估计就是政府了，深圳市在一两年前就推出了公共单车，在地铁站、公园附件都设置了固定存取点，要到政府部门缴纳押金才能领取IC卡，凭卡存取单车，充值也要到专门的地方进行，所以虽然我经常带着虾米路过存车点，但嫌麻烦没有去办理过这玩意，据我的观察，也没有几个人在用这些公共单车。
互联网企业搞的共享单车，一出来就解决了上述公共单车的几个痛点，随处存取、手机操作，简单想一下，算是将维护的责任丢给了公众，靠大家自觉将单车停在适合的地点，摆放整齐。
在这里又有两种思路，一种是大政府的概念，要求政府把什么都管起来，对于共享单车来说，就是要求城管什么的，维护这些单车的停放；另外一种思路是，政府你千万别管天管地管空气，俺们自己能管好，就像去香港行山，一路见不到垃圾桶，游客都是自己拿个袋子把垃圾随身带着，垃圾不落地清洁人员就省事多了，政府也就没借口多收税啦。
两种思路，各有利弊。

3. 有了共享单车之后，不怕丢单车了；

有一天在电梯里，听到同事说起个有趣的论调：自从有了共享单车，我再也不怕自家的单车丢失啦，你想啊，路边随便5毛钱就可以骑车了，谁还去买二手单车啊，自然也没有人去偷单车来卖了。
想起以前我小时候，深圳偷单车有多猖獗，就觉得现在这个共享单车也不是一无是处，好多人都觉得自家买的代步单车是累赘啦，恨不得能卖给租给小黄小蓝，省心省事。

4. 昧了共享单车之后，更怕丢单车了；

共享单车也有烦心事，小黄这种笨单车，没定位系统没有动态密码，一旦被别有用心的人利用，几乎就收不回去，好多人特别是小孩，破解之后就据为己有了，甚至对抗小黄的工作人员，阻挠回收单车。
但是昧了共享单车之后，会不会更怕丢单车了呢，因为这始终有标记是别人的东西，小黄的工作人员可以光明正大地回收，其他用户可以名正言顺地骑走，就算你拿锁头把它锁起来，遇到其他无赖当面敲了你的锁，你又能找谁说理去？

打碎花瓶之后

发表于 2017-05-20 | 分类于 家庭生活

今天下午去东门买了校服回来，大人小孩都比较兴奋，虾米拿了小皮球想玩。我正准备检查出门前RO挂机的效果，所以让他和婆婆在客厅玩一下，没多久就听到“哐当”玻璃碎掉的声音。我和老婆赶紧大叫一声：“先别动！”，出来一看，花瓶被打碎了。
这时虾米已经吓得哭了出来，一个劲地说“我不是故意的”，老婆和婆婆安慰他说没有关系，但我觉得虾米从小就看过《大卫，不可以》，里面也有在家里玩耍打碎花瓶的情节，所以要让他明白是这做了错事。
